Derivative Rules

Expanded Explanation of the Derivative Rules

1. Power Rule

If f(x) = xn, then f'(x) = n·xn-1.

This rule applies when you have a variable raised to a constant power. For example, if f(x) = x3, then f'(x) = 3x2. This rule is fundamental in calculus and simplifies finding the derivative of polynomial functions.

2. Constant Rule

If f(x) = c, where c is a constant, then f'(x) = 0.

A constant function is flat, meaning it has no rate of change, so its derivative is zero. Example: f(x) = 7 → f'(x) = 0.

3. Constant Multiple Rule

If f(x) = c·g(x), then f'(x) = c·g'(x).

You can factor out constants when differentiating. For example, if f(x) = 5x2, then f'(x) = 5·(2x) = 10x.

4. Sum Rule

If f(x) = g(x) + h(x), then f'(x) = g'(x) + h'(x).

You can take the derivative of each term separately. Example: f(x) = x2 + x → f'(x) = 2x + 1.

5. Difference Rule

If f(x) = g(x) - h(x), then f'(x) = g'(x) - h'(x).

Similar to the sum rule but with subtraction. For instance, f(x) = x3 - x → f'(x) = 3x2 - 1.

6. Product Rule

If f(x) = g(x)·h(x), then f'(x) = g'(x)·h(x) + g(x)·h'(x).

This rule is used when multiplying two functions. For example, if f(x) = x2·sin(x), then f'(x) = 2x·sin(x) + x2·cos(x).

7. Quotient Rule

If f(x) = g(x)/h(x), then f'(x) = (g'(x)·h(x) - g(x)·h'(x)) / [h(x)]2.

Used when one function is divided by another. For instance, if f(x) = x / (x+1), then apply the rule to get the derivative.

8. Chain Rule

If f(x) = g(h(x)), then f'(x) = g'(h(x))·h'(x).

This rule is used for composite functions. For example, if f(x) = (3x+1)2, then f'(x) = 2·(3x+1)·3 = 6·(3x+1).

MCQ 1

What is the derivative of f(x) = x5?

Explanation: Using the Power Rule: derivative of xn is n·xn-1, so derivative of x5 is 5x4.

MCQ 2

If f(x) = 3x, what is f'(x)?

Explanation: The derivative of a linear function f(x) = ax is simply the coefficient a, so derivative is 3.

MCQ 3

Find the derivative of f(x) = x2 + x

Explanation: f(x) = x2 + x ⇒ f'(x) = 2x + 1 by applying Power Rule to each term.

MCQ 4

What is the derivative of a constant function?

Explanation: The derivative of a constant is always 0 by the Constant Rule.

MCQ 5

What is the derivative of f(x) = x3 - x?

Explanation: Derivative of x3 is 3x2 and derivative of -x is -1, so total derivative is 3x2 - 1.

MCQ 6

If f(x) = 2x2 and g(x) = x3, find d/dx[f(x) + g(x)]

Explanation: Use the Sum Rule: derivative of 2x2 is 4x, derivative of x3 is 3x2, so sum is 4x + 3x2.

MCQ 7

Find the derivative of f(x) = (x2)(sin x)

Explanation: Using Product Rule: f(x) = u·v, f'(x) = u'·v + u·v'. Here, u = x2 and v = sin x, so f'(x) = 2x·sin x + x2·cos x.
